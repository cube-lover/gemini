## ⚙️ 配置说明（简版）

- `api_url`：填 Flow2API 的 OpenAI 兼容接口地址（/v1/chat/completions）
- `apikey`：填你自定义的密钥（例如: han1234）
- `model`：默认 6 个可选模型之一
- `custom_model`：可选，自定义模型ID（你有私有/转发模型名就填这里）
